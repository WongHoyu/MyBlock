## 每日一命令之cp
### 1. 递归复制 
```
cp -r * 
```
### 2. 对 xxx 创建快捷方式:(展示方式) xxx -> yyy
``` 
cp -s xxx yyy
```
### 3. 如果目录 y 存在,则 cd y 后存在 test.sh
```
cp test.sh y
```
### 4. 如果目录 y 不存在,则 y 是 test 的复制，**不会生成目录**
```
cp test.sh y 
```
### 5.  为每个已存在的目标文件创建备份,相当于 cp -dR --preserve=all,复制后的文件的创建时间不更新
```
cp -a xxx yyy
```
### 6. cp -b 类似于 --backup
